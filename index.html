Why Not Pointers..?

## Familiar Syntax It's the syntax you love, reimagined for the most advanced runtime available today. A simple example: ```cs greet(userId) { user = db.getUser(userId); return "Hello \{user->name}!"; } process.stdout << *greet("alex") << '\n'; ``` This compiles to the following ES6 code: ```js async function greet(userId) { const user = db.getUser(userId); return `Hello ${(await user).name}`; } greet("alex").then((a$) => console.log(a$)); ```
